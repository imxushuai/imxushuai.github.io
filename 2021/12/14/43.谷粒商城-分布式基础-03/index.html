JRS-303校验

JSR-303 是 Java EE 6 中的一项子规范,叫做 Bean Validation,官方参考实现是hibernate Validator。

之前写另一篇的文章的时候，也有用到Spring MVC的相关注解进行参数校验，但没这么全面，我重新单开了一篇文章把JRS-303重新记录一下。

文章链接：

  1. 实体类的字段添加注解，表示其字段的校验规则

  2. 在gulimall-common中创建校验分组：AddGroup、UpdateGroup和UpdateStatusGroup

    • AddGroup

    • UpdateGroup

    • UpdateStatusGroup

  3. 在gulimall-common中创建自定义校验注解

  4. 在gulimall-common中创建自定义校验器，用于对自定义注解标注的字段进行校验

  5. 在gulimall-product的BrandController中的API上添加@Validated注解标注进行参数校验以及所使用的校验分组

  6. 在gulimall-product中新增统一异常处理类

分布式基础篇总结

  1. 分布式基础概念

    微服务、注册中心、配置中心、远程调用、Feign、网关

  2. 基础开发

    Spring Boot、Spring Cloud、Mybatis-Plus、Vue组件、阿里云存储

  3. 环境

    虚拟机、Linux、Docker、MySQL、Redis、人人开源相关项目

  4. 开发规范

    • 数据校验JSR303、全局异常处理、全局统一返回、全局跨域处理
    • 枚举状态、业务状态码、VO/TP/PO划分、逻辑删除（假删除）
    • Lombok
